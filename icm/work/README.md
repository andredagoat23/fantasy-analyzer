# work/ — Working Artifacts (Layer 4)

Scratch space for the CURRENT task only. Ephemeral by design.

Stage 01 writes `diagnosis.md` here; Stage 02 may write `plan.md`; Stage 04 may drop verification
output. The next task overwrites or clears them — nothing here is durable.

**Durable knowledge does NOT live here.** If something learned in a task should survive (a new fact,
a fixed assumption, a convention), it goes in `../reference/` (or `../reference/lessons.md`) during
Stage 05, not in this folder.
